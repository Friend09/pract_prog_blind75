üéØ Algorithm Overview

The Count Islands algorithm uses Depth-First Search (DFS) to identify and count connected components of land cells in a binary matrix. Each island is a group of adjacent land cells (1s) connected horizontally or vertically.

Main Function: count_islands

from typing import List def count_islands(matrix: List[List[int]]) -> int: # Handle empty matrix edge case if not matrix: return 0 # Initialize island counter count = 0 # Iterate through each cell in the matrix for r in range(len(matrix)): for c in range(len(matrix[0])): # If a land cell is found, perform DFS to explore the full # island, and include this island in our count. if matrix[r][c] == 1: dfs(r, c, matrix) count += 1 return count

DFS Helper Function

def dfs(r: int, c: int, matrix: List[List[int]]) -> None: # Mark the current land cell as visited by setting it to -1 matrix[r][c] = -1 # Define direction vectors for up, down, left, and right dirs = [(-1, 0), (1, 0), (0, -1), (0, 1)] # Recursively call DFS on each neighboring land cell to continue # exploring this island for d in dirs: next_r, next_c = r + d[0], c + d[1] if is_within_bounds(next_r, next_c, matrix) and matrix[next_r][next_c] == 1: dfs(next_r, next_c, matrix)

Bounds Checking Helper

def is_within_bounds(r: int, c: int, matrix: List[List[int]]) -> bool: # Check if row and column indices are within matrix boundaries return 0 <= r < len(matrix) and 0 <= c < len(matrix[0])

Time & Space Complexity

Aspect Complexity Explanation
Time Complexity O(M √ó N) Each cell is visited at most twice: once in the main loop, once in DFS
Space Complexity O(M √ó N) Worst case: DFS recursion stack for a matrix that's all land
Space (Modified Matrix) O(1) If we're allowed to modify input matrix (using -1 for visited)

üß† Key Insights

Why DFS Works Perfect Here

DFS naturally explores all connected components (islands) by following the "path" of connected land cells. Once we mark a cell as visited, we never count it again, ensuring each island is counted exactly once.

Memory Optimization

By modifying the input matrix (changing 1s to -1s), we avoid using extra space for a visited array. This is acceptable since we're just counting islands, not preserving the original matrix.

Alternative Approaches
  • BFS: Could use a queue instead of recursion
  • Union-Find: More complex but useful for dynamic connectivity
  • Iterative DFS: Using explicit stack to avoid recursion depth limits

üéØ Common Use Cases

  • Geographic Analysis: Counting landmasses, lakes, or regions
  • Image Processing: Connected component analysis
  • Network Analysis: Finding disconnected subgraphs
  • Game Development: Terrain analysis, flood fill algorithms

üéØ Summary

The Count Islands algorithm efficiently identifies connected components in a binary matrix using DFS. It's a fundamental graph traversal problem with wide applications in computer science, from image processing to geographic analysis.

Key takeaways:

  • DFS naturally handles connected component detection
  • Modifying input matrix saves space (O(1) extra space)
  • Time complexity is optimal: O(M√óN) - each cell visited at most twice
  • Handles all edge cases: empty matrix, all water, all land
